# 3irobotix
## 米家APP扫地机器人配网自动化
* 1.打开米家APP
* 2.点击右上角 "+" 加号，点击选择“添加设备”
* 3.搜索设备页添加设备：
	1）自动搜索：小米扫拖机器人2 Lite
	2）手动输入：
		点击右上角搜索按钮
		senkey:小米扫拖机器人2 Lite
		点击搜索历史出现的 小米扫拖机器人 2Lite 
		跳转页面勾选：已确认上述操作
		下一步
* 选择路由器
	连接其它路由器
	选择：common_para中提供的new_ssid一致的WiFi，点击
	下一步
	跳转到配网页面，等待完成（5秒）
	自动跳转到选择房间：点击下一步
	设备重命名：
		点击输入框
		点击 x 先清空
		sendkey: auto_260v_1
		下一步
		开始体验
* 同意并继续
	sleep（2）
* 清扫指引：
	向左滑动2次
	确定
	下滑屏幕
*进入Rel-test房间
	长按 auto_260v_1 删除设备
	弹窗确定
